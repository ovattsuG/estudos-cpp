# Conversor de Moedas em C++

## Descrição
Este programa em C++ converte valores de diferentes moedas internacionais para Real (R$). 
O usuário escolhe a moeda de origem, informa o valor e o programa calcula o equivalente em reais.

## Objetivo
Praticar o uso de:
- Estrutura switch
- Entrada e saída de dados
- Operações matemáticas
- Formatação de saída

## Conceitos utilizados
- std::cin e std::cout
- Estrutura switch / case
- Variáveis do tipo int e double
- Operações matemáticas
- Formatação com std::fixed e std::setprecision

## Moedas disponíveis
1 - USD (Dólar Americano)  
2 - EUR (Euro)  
3 - JPY (Iene Japonês)  
4 - GBP (Libra Esterlina)  
5 - CHF (Franco Suíço)  

## Como funciona
O programa:
1. Mostra um menu com opções de moedas
2. O usuário escolhe uma opção
3. O usuário digita o valor que deseja converter
4. O programa aplica uma taxa de câmbio fixa
5. Exibe o valor convertido em reais (R$)

## Exemplo de uso
### Selecione qual moeda deseja converter para R$
1 - USD
2 - EUR
3 - JPY
4 - GBP
5 - CHF

1

Digite o valor que deseja converter:
10

O valor em reais = R$52.20

## Observação
As taxas de câmbio utilizadas são valores fixos apenas para fins de estudo e podem não refletir o valor real do mercado.
