# Conversor de Temperatura em C++

## Descrição
Este programa em C++ converte temperaturas entre Celsius e Fahrenheit. 
O usuário escolhe o tipo de conversão e informa o valor da temperatura, e o programa exibe o resultado convertido.

## Objetivo
Praticar o uso de:
- Estrutura switch
- Entrada e saída de dados
- Operações matemáticas
- Manipulação de caracteres
- Biblioteca <cctype>

## Conceitos utilizados
- std::cin e std::cout
- switch / case
- Variáveis inteiras
- Operações matemáticas
- Função std::toupper()
- Biblioteca <cctype>

## Como funciona
O programa pede ao usuário para escolher:
- C → Converter Celsius para Fahrenheit
- F → Converter Fahrenheit para Celsius

Depois o usuário digita a temperatura e o programa realiza a conversão e mostra o resultado.

## Fórmulas utilizadas
Celsius → Fahrenheit:
F = (C × 9/5) + 32

Fahrenheit → Celsius:
C = (F − 32) × 5/9

## Exemplo de uso
Digite C para converter Celsius para Fahrenheit ou F para converter Fahrenheit para Celsius:
C

Digite o valor que deseja converter:
30

30C em Fahrenheit = 86F
