# Tabuada em C++

## Descrição
Este programa em C++ gera a tabuada de multiplicação de 1 até 10 utilizando estruturas de repetição aninhadas (for dentro de for).

## Objetivo
Praticar o uso de loops, principalmente o uso de for aninhado, e entender melhor como funcionam repetições dentro de repetições.

## Conceitos utilizados
- Estrutura de repetição for
- Loop aninhado (for dentro de for)
- Operações matemáticas
- Saída de dados com std::cout
- Estrutura básica de programa em C++

## Como funciona
O programa utiliza dois loops:
- O primeiro loop controla o número da tabuada
- O segundo loop controla o número que será multiplicado
- Para cada combinação, o programa imprime o resultado da multiplicação

## Exemplo de saída
1 X 1 = 1  
1 X 2 = 2  
1 X 3 = 3  
...  

2 X 1 = 2  
2 X 2 = 4  
2 X 3 = 6  
...  

Até a tabuada do 10.
