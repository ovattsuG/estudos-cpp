# Cálculo de Fatorial em C++

## Descrição
Este programa em C++ calcula o fatorial de um número inteiro informado pelo usuário. 
O programa também verifica se o número é negativo, pois não existe fatorial para números negativos.

## Objetivo
Praticar o uso de:
- Estruturas de repetição (for)
- Estruturas condicionais (if / else)
- Operações matemáticas
- Entrada e saída de dados

## Conceitos utilizados
- Variáveis inteiras
- std::cin e std::cout
- Loop for
- Estrutura condicional if / else
- Operações matemáticas
- Lógica de programação

## Como funciona
O programa pede ao usuário um número inteiro.
Se o número for maior ou igual a zero, o programa calcula o fatorial utilizando um loop.
Se o número for negativo, o programa informa que não existe fatorial para números negativos.

O fatorial de um número é o produto de todos os números inteiros de 1 até esse número.

Exemplo:
5! = 5 × 4 × 3 × 2 × 1 = 120

## Exemplo de uso
Digite o numero para calcular o fatorial:
5

Fatorial de 5 = 120

Digite o numero para calcular o fatorial:
-3

Nao existe fatorial de numero negativo
