# Par ou Ímpar em C++

## Descrição
Este é um programa simples em C++ que lê um número inteiro digitado pelo usuário e informa se o número é par ou ímpar.

## Objetivo
Este exercício foi feito para praticar conceitos básicos de programação em C++.

## Conceitos utilizados
- Entrada de dados (std::cin)
- Saída de dados (std::cout)
- Estrutura condicional (if / else)
- Operador módulo (%)
- Variáveis inteiras
- Estrutura básica de um programa em C++

## Como funciona
O programa pede para o usuário digitar um número.  
Depois ele verifica se o número é divisível por 2.  
Se for divisível por 2, o número é par.  
Caso contrário, o número é ímpar.

## Exemplo de uso
Digite um valor para descobrir se é par ou ímpar:
10
10 é Par!

Digite um valor para descobrir se é par ou ímpar:
7
7 é impar!
