# Maior e Menor Valor em C++

## Descrição
Este programa em C++ lê três números inteiros informados pelo usuário e determina qual é o maior valor e qual é o menor valor entre eles.

## Objetivo
O objetivo deste exercício é praticar lógica de programação, comparações entre valores e uso de estruturas condicionais (if / else).

## Conceitos utilizados
- Variáveis inteiras
- Entrada de dados com std::cin
- Saída de dados com std::cout
- Estruturas condicionais (if / else if / else)
- Operadores de comparação
- Lógica para encontrar maior e menor valor

## Como funciona
O programa solicita ao usuário três números inteiros.
Depois ele compara os valores para encontrar:
- O maior número
- O menor número

Os resultados são exibidos na tela ao final do programa.

## Exemplo de uso
Insira tres valores para descobrir qual o maior:
10 5 20

O maior valor = 20
O menor valor = 5
