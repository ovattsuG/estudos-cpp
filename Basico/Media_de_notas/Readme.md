# Média de Notas em C++

## Descrição
Este programa em C++ lê três notas de um aluno, calcula a média e informa se o aluno foi aprovado ou reprovado com base em uma nota de corte.

## Objetivo
Praticar entrada de dados, operações matemáticas, uso de variáveis e estruturas condicionais em C++.

## Conceitos utilizados
- Variáveis do tipo double e int
- Entrada de dados com std::cin
- Saída de dados com std::cout
- Operações matemáticas
- Estrutura condicional if / else
- Cálculo de média

## Como funciona
O programa solicita ao usuário três notas.
Em seguida, calcula a média das notas e compara com a nota de corte (7).
Se a média for maior ou igual a 7, o aluno está aprovado.
Caso contrário, o aluno está reprovado.

## Exemplo de uso
Insira as notas do Aluno:
7.5
6.0
8.0

A media das notas = 7.16
Aluno Aprovado!
