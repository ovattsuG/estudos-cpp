# Contador de Vogais em C++

## Descrição
Este programa em C++ lê uma palavra digitada pelo usuário e conta quantas vogais existem nela.

## Objetivo
Praticar manipulação de strings em C++, uso de arrays, loops aninhados e a biblioteca <cstring>.

## Conceitos utilizados
- Arrays de char
- Strings em C
- Biblioteca <cstring>
- Função strlen()
- Estruturas de repetição (for)
- Loop aninhado
- Estruturas condicionais (if)
- Entrada e saída de dados (std::cin e std::cout)

## Como funciona
O programa:
1. Pede ao usuário uma palavra
2. Percorre cada letra da palavra
3. Compara cada letra com um vetor contendo as vogais
4. Conta quantas vogais existem na palavra
5. Mostra o resultado na tela

## Exemplo de uso
Insira uma palavra para saber o numero de vogais dela:
programacao

A palavra programacao contem 5 vogais!!
